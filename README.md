Travel Booking System

Wayfarer is a professional travel management platform designed to streamline the process of managing bookings, itineraries, and travel expenses through a modern and intuitive interface.

## Overview

This project focuses on providing a comprehensive suite of tools for travelers to organize their journeys. It features a responsive dashboard, detailed booking management, and integrated trip planning capabilities.

## Key Features

- Dashboard
  Detailed overview of active bookings, total profit, and upcoming trip schedules with visual trends and quick actions.

- Booking Management
  Interface for tracking flights, hotels, and car rentals. Includes status indicators and filtering capabilities for efficient management.

- Itinerary Builder
  Tool for organizing multiple bookings into cohesive trips. Provides route visualization and chronologically ordered travel plans.

- Expense Tracker
  System for logging travel-related costs. Supports categorization and links expenses directly to specific itineraries for better budget control.

- Traveler Profiles
  Secure storage for individual traveler information, including passport data, seat preferences, and loyalty program details.

- Analytics and Reports
  Visual breakdowns of profit patterns, booking distributions, and travel frequency to help optimize travel budgets.

## Frontend Technology Stack

- React 19
  Using the latest React features for building a dynamic and responsive user interface.

- TypeScript
  Ensuring type safety and maintainability across the entire application codebase.

- Vite
  Utilizing a high-performance build tool for a fast development experience and optimized production bundles.

- Tailwind CSS
  Implementing a modern, utility-first styling approach for a polished and consistent design language.

- shadcn/ui and Radix UI
  Leveraging accessible and highly customizable UI components for a premium user experience.

- Zustand
  Lightweight state management for efficient data handling across various application sections.

- Wouter
  Minimalist and fast routing solution for seamless navigation between pages.

## Project Structure

client/
src/
pages/ - Individual page views for each application section
components/ - Reusable UI elements and interface blocks
lib/ - Client-side utilities and global state stores
hooks/ - Custom React hooks for shared logic
App.tsx - Main routing and layout configuration
public/ - Static assets and image files

## Getting Started

1. Install dependencies
   Using pnpm to manage project packages:
   pnpm install

2. Run the development server
   Launch the frontend application locally:
   pnpm dev

The application will be accessible at http://localhost:3000
